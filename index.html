HTML Tags

HTML Tags are fundamental elements used to structure and format content on web pages. They provide instructions to web browsers on how to render text, images, links, and other media.

TAG NAME DESCRIPTION
<html> Html document tag This tag contains all of the HTML elements.
<head> Head First section of your document, which is not display on the page, but contains metadata, and/or information about the page.
<title> Title Html elemente that specifies the title of a webpage, this tag is very important for the SEO.
<body> Body This element contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc. Note: There can only be one element in an HTML document.
<h1> headings Defines the most important heading. <h6> defines the least important heading. Note: Only use one <h1> per page - this should represent the main heading/subject for the whole page.
<a> Anchor Creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address.
<ul/ol> Unordered list and Ordered list The ol element is used when the list is ordered and the ul element is used when the list is unordered.
<p> Paragraphs this tag is a fundamental element used for creating paragraphs in web development. It helps structure content, separating text into distinct blocks.
<img> Image It is used to embed an image in an HTML page. Images are not technically inserted into a web page; images are linked to web pages.
<table> Table HTML table tag is used for represent structured data grid organized into rows and columns.

Semantic HTML Tags

TAG NAME DESCRIPTION
<header> Header This tag represent introductory content
<nav> Navigation Used for navigations menus and links.
<main> Main Contains the core information of a page.
<section> Section defines the section of documents such as chapters, headers, footers, or any other sections.
<article> Article This tag specifies independent, self-contained content.
<figure> Figure Used for embedding images, charts, or diagrams, along with a caption
<footer> Footer This tag contains contact details, and copyright information.
